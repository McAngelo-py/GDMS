# GDMS - Greenguy Digital Marketing Studio

A professional community platform for video editors, designers, photographers, animators, VAs, and content creators.

## 🚀 Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Project Structure**: Organized codebase with separate HTML, CSS, and JS files.
- **Interactive Loader**: Professional opening animation with brand logo.
- **Hero Section**: High-impact background with brand messaging and call-to-actions.
- **Courses Section**: Dedicated standalone page for educational content.
- **Community Integration**: Quick links to Facebook Messenger and Discord.

## 📂 Project Structure

- `index.html`: Main landing page.
- `courses.html`: Dedicated courses and learning page.
- `css/style.css`: All styling, including animations and responsive breakpoints.
- `js/script.js`: Core interactivity and loader logic.
- `assets/images/`: Images, logos, and hero backgrounds.

## 🛠️ Technologies Used

- HTML5 & CSS3 (Flexbox/Grid, Custom Properties)
- JavaScript (Intersection Observer, Event Listeners)
- [Font Awesome](https://fontawesome.com/) (Icons)
- [Google Fonts](https://fonts.google.com/) (Inter typeface)

## 📝 Notes for Developers

- **Loader**: The loader uses a 1000ms delay by default in `script.js` to ensure a smooth transition.
- **Hero Background**: The hero image is set as a background in `style.css` with a semi-transparent overlay for readability.
- **Animations**: Uses native CSS animations and `Intersection Observer` for reveal-on-scroll effects.
