# GDMS - Greenguy Digital Marketing Studio

A professional community platform for video editors, designers, photographers, animators, VAs, and content creators.

## 🚀 Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop viewing.
- **Project Structure**: Organized codebase with separate HTML, CSS, and JS files.
- **Interactive Loader**: Professional opening animation with brand logo.
- **Hero Section**: High-impact background with brand messaging and call-to-actions.
- **Courses Page**: Dedicated standalone page for educational content.
- **Team Page**: Beautiful standalone page with 3D flip effect cards.
- **Admin Dashboard**: Isolated admin page with purple theme and stats.
- **Community Integration**: Quick links to Facebook Messenger and Discord.
- **Cursor Effects**: Subtle dot trail cursor following animation.
- **Reveal Animations**: Scroll-triggered reveal effects for better user experience.
- **Glassmorphism Design**: Modern frosted glass effects on navigation.

## 📂 Project Structure

- `index.html`: Main landing page.
- `courses.html`: Dedicated courses and learning page.
- `team.html`: Standalone team page with flip cards.
- `admin.html`: Isolated admin dashboard.
- `css/style.css`: All styling, including animations and responsive breakpoints.
- `js/script.js`: Core interactivity, navigation, and scroll effects.
- `js/cursor.js`: Custom cursor dot trail animation.
- `assets/images/`: Images, logos, and hero backgrounds.

## 🛠️ Technologies Used

- HTML5 & CSS3 (Flexbox/Grid, Custom Properties)
- JavaScript (Intersection Observer, Event Listeners, DOM Manipulation)
- [Font Awesome](https://fontawesome.com/) (Icons)
- [Google Fonts](https://fonts.google.com/) (Inter typeface)

## 🎨 Design Highlights

### Design System
- **CSS Custom Properties**: Centralized variables for consistent theming.
- **Professional Color Palette**: Green primary with complementing gradients.
- **Subtle Animations**: Smooth transitions and hover effects throughout.

### Interactive Features
- **3D Flip Cards**: Team members cards flip on hover to show bios.
- **Custom Cursor**: Subtle green dot trail follows the cursor.
- **Scroll Reveals**: Sections animate in as you scroll down.
- **Glassmorphism Nav**: Frosted glass navigation with backdrop blur.

## 📝 Notes for Developers

- **Loader**: The loader uses a 1000ms delay by default in `script.js` to ensure a smooth transition.
- **Hero Background**: The hero image is set as a background in `style.css` with a semi-transparent overlay for readability.
- **Animations**: Uses native CSS animations and `Intersection Observer` for reveal-on-scroll effects.
- **Isolated Admin Page**: `admin.html` has its own scoped CSS to prevent conflicts with main styles.
- **Flip Cards**: Team cards use CSS 3D transforms and perspective for the flip effect.
- **Cursor Effect**: `cursor.js` dynamically creates and animates dot elements.
